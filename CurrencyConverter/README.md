# Currency Converter Using API

## Description

Java based currency converter application using live exchange rate API.

## Features

- User selects base currency
- User selects target currency
- Takes amount input
- Fetches live exchange rates
- Displays converted amount


## Technologies

- Java
- HTTP Client
- REST API
- JSON Parsing


## API Used

ExchangeRate API


## Run Project

Compile:

javac Main.java CurrencyConverterAPI.java


Run:

java Main


## Example

Input:

USD
INR
100


Output:

100 USD = 8300 INR
