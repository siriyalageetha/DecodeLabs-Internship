# Student Grade Calculator

## Description
A Java console application that calculates:

- Total Marks
- Average Marks
- Percentage
- Grade

## Features

- Multiple subject support
- Input validation
- Percentage calculation
- Automatic grade assignment
- Clear result display

## Grade Criteria

| Percentage | Grade |
|------------|--------|
| 90-100 | A+ |
| 80-89 | A |
| 70-79 | B |
| 60-69 | C |
| 50-59 | D |
| Below 50 | F |

## How to Run

Compile:

```bash
javac Calculator.java
```

Run:

```bash
java Calculator
```

## Sample Output

```text
Total Marks : 440
Average Marks : 88.00
Percentage : 88.00%
Grade : A
```
