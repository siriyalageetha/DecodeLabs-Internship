Number Guessing Game
A Java console-based game where the player tries to guess a randomly generated number between 1 and 100.

Features
Random number generation
High/Low hints
Limited attempts
Multiple rounds
Score tracking
Technologies Used
Java
Random Class
Scanner Class
Loops and Conditionals
How to Run
Compile
javac NumberGame.java
Run
java NumberGame
